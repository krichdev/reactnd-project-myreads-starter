# MyReads Book Tracking App - Udacity React Project

This is the first project for the Udacity React Nanodegree program. This app helps track books by placing them into different shelves ('Want to Read', 'Currently Reading', and 'Read').

create-react-app was used to create this project.

## Get Started
Run `npm install` or `yarn` from inside project directory.

Then use `npm start` or `yarn start` to launch the MyReads app.
